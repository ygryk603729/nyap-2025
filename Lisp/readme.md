# Рекурсивный вычислитель S-выражений (Common Lisp)

## Постановка задачи
Программа читает файл `input.txt`, содержащий арифметические S-выражения Common Lisp (по одному на строку).  
Рекурсивно парсит и вычисляет каждое выражение (+, -, *, /), записывает результат в `output.txt` по одному на строку.

Поддерживаемые операторы: +, -, *, / (с любым количеством аргументов).  
Вложенность произвольная.

## Тестовые файлы
- `input1.txt` — 5 простых выражений (маленький тест)
- `input2.txt` — 20 выражений с вложенностью (средний тест)

## Запуск программы
1. Установить SBCL (Windows installer с sbcl.org)
2. В папке с кодом выполнить:
   ```powershell
   sbcl --script calc.lisp -i input1.txt -o output1.txt
