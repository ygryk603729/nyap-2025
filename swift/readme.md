# Файловый сервер и клиент (Swift)

## Постановка задачи
Реализован простой TCP-файловый сервер и клиент на языке Swift.  
Сервер слушает подключения на указанном порту и по простому HTTP-подобному GET-запросу отправляет запрашиваемый файл клиенту.  
Клиент запрашивает файл по его имени у сервера и сохраняет полученное содержимое в указанный файл.

Клиент и сервер могут быть запущены на разных языках (по ТЗ допускается), но в этой реализации оба написаны на Swift.

## Структура проекта
- `server.swift` — код сервера
- `client.swift` — код клиента
- В папке сервера должны лежать файлы, которые клиент будет запрашивать (они играют роль "input-файлов")

## Тестовые файлы (для сервера)
Создайте в папке, где запускается сервер, три файла для демонстрации:
- `file1.txt` — маленький текстовый файл (например, несколько строк текста)
- `file2.txt` — файл среднего размера (несколько абзацев или килобайт текста)

## Особенности реализации

Использованы прямые TCP-сокеты из модуля Foundation (стандартная библиотека Swift)
Простой HTTP-подобный протокол (GET-запрос и ответ с заголовками)
Поддержка передачи файлов любого размера (чтение/запись потоками)
Ручной парсинг аргументов командной строки
Полная совместимость с Swift для Windows
