# Рекурсивный DFS в графе (Scala)

## Постановка задачи
Программа выполняет рекурсивный поиск в глубину (DFS) в неориентированном графе, заданном матрицей смежности.  
Находит любой путь от стартовой вершины до конечной и записывает его в `output.txt` (вершины через " -> ").  
Если пути нет — пишет "Путь не найден".

## Формат входных данных
- Первые n строк — матрица смежности (n x n, 0/1, разделитель пробелы)
- Последняя строка — два числа: стартовая и конечная вершина (нумерация с 0)

## Тестовые файлы
- `input1.txt` — маленький граф с коротким путём
- `input2.txt` — граф с циклом и несколькими путями
- `input3.txt` — большой граф (50+ вершин)

## Запуск программы (Scala)
```powershell
scala dfs.scala                    # input.txt → output.txt
scala dfs.scala -i input1.txt -o output1.txt
```

## Особенности реализации

Рекурсивный DFS с накоплением пути
Прерывание рекурсии через исключение при нахождении пути
Неизменяемые коллекции и функциональный стиль
Полностью на Scala 3
