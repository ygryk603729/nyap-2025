# "Новые языки программирования" (2025)

Темы задач (по ТЗ):
1. Многопоточность
2. Алгоритмическая задача с рекурсией
3. Конвертация данных из одного формата в другой
4. Клиент-серверное взаимодействие
5. Парсинг данных с регулярными выражениями

Ввод/вывод — через файлы (input.txt / output.txt, иногда дополнительные файлы).  
По каждой задаче подготовлены:
- Исходный код
- README.md с постановкой задачи, форматом данных и инструкцией по запуску
- Тестовые файлы (3 сценария: маленький, средний, большой/edge-case)
- Видео-объяснение (10–15 минут)

Все проекты лежат в отдельных папках.

## Общие замечания
- Все программы используют файлы ввода/вывода (input.txt, patterns.txt и т.д.).
- По каждой задаче есть несколько тестовых сценария для демонстрации в видео.
- Код написан идиоматично для каждого языка, с акцентом на "фишку" языка по теме.
- Видео-объяснения включают: постановку задачи, почему выбран язык, разбор кода, демонстрацию запуска на 3 тестах.
- Установка языков описана в отдельных README или в видео.

## Требования к сдаче (по ТЗ)
- Исходный код — ссылка на Git-репозиторий (рекомендую создать один репозиторий с папками по языкам)
- Документация — структура входных данных (в README.md каждой папки)
- Видео-файлы с пояснением работы программы и примерами запусков
