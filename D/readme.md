# Параллельная фильтрация простых чисел (D)

## Постановка задачи
Программа читает `input.txt`:
- Первая строка — число N
- Вторая строка (или несколько) — числа от 2 до N

Отфильтровывает простые числа с помощью многопоточности, сортирует по возрастанию и записывает в `output.txt` по одному на строку.

## Тестовые файлы
- `input1.txt` — N = 100, числа от 2 до 100
- `input2.txt` — N = 1000, числа от 2 до 1000

## Запуск
1. Скомпилировать:
   ```powershell
   dmd prime_filter.d -of=prime_filter.exe
   ```
