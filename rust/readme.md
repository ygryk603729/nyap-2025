# Параллельный подсчёт частоты слов

## Постановка задачи
Программа читает текстовый файл `input.txt`, подсчитывает частоту встречаемости каждого слова  
(игнорируя регистр, знаки препинания и другие неалфавитно-цифровые символы),  
и записывает результат в `output.txt` в формате: слово количество

Строки отсортированы по убыванию количества, при равенстве — по алфавиту слова.

## Формат входных данных
- Файл `input.txt` — произвольный текст в кодировке UTF-8.
- Слова разделяются пробелами, табуляцией, переводами строк.
- Регистр игнорируется.
- Знаки препинания и специальные символы удаляются.

## Тестовые файлы
В папке находятся:
- `input1.txt` — простой английский текст
- `input2.txt` — русский текст со знаками препинания
- `input3.txt` — большой текст (~50 000 слов) для демонстрации параллелизма

## Запуск программы (Rust)
1. Убедитесь, что установлен Rust (rustc и cargo).
2. Поместите один из `input*.txt` в папку с исполняемым файлом и переименуйте в `input.txt`  
   или передайте путь как аргумент.
3. Выполните:
   ```bash
   cargo run --release

или, если скомпилировали вручную:

wordcount.exe [путь_к_input.txt] [путь_к_output.txt]

Результат будет в output.txt.
