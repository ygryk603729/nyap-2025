# Конвертация CSV → JSON

## Постановка задачи
Программа читает CSV-файл (`input.txt`), где первая строка — заголовки колонок,  
преобразует данные в массив JSON-объектов и записывает в `output.txt` (pretty-printed).

Числовые значения сохраняются как числа, остальные — как строки.

## Формат входных данных
- Файл в кодировке UTF-8
- Разделитель — запятая
- Первая строка — названия колонок
- Пустые файлы и файлы без данных обрабатываются корректно

## Тестовые файлы
- `input1.txt` — простая таблица людей
- `input2.txt` — товары с ценами и булевыми значениями
- `input3.txt` — большая таблица (1000+ строк)
